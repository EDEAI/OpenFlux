## 工具列表

### 文件与编码

| 工具 | 说明 |
|------|------|
| **filesystem** | 文件读写、创建、删除、搜索 |
| **opencode** | 代码编辑辅助 |
| **process** | 运行系统命令 |

### Web

| 工具 | 说明 |
|------|------|
| **browser** | 浏览器自动化（打开网页、点击、截图、表单等） |
| **web_search** | Web 搜索（支持区域、语言、时间过滤） |
| **web_fetch** | 网页内容抓取（URL → Markdown/纯文本） |

### 系统控制

| 工具 | 说明 |
|------|------|
| **windows** | 剪贴板、窗口管理、启动应用、PowerShell、COM 自动化 |
| **desktop** | 桌面控制 |

### 调度与工作流

| 工具 | 说明 |
|------|------|
| **scheduler** | 定时任务（cron / 间隔 / 一次性） |
| **workflow** | 多步骤工作流（创建、执行、管理） |

### 办公与通信

| 工具 | 说明 |
|------|------|
| **office** | 办公工具 |
| **email** | 邮件收发（POP3 + SMTP） |
| **notify_user** | 向用户推送通知 |

### 协作

| 工具 | 说明 |
|------|------|
| **spawn** | 派生子 Agent 执行子任务 |
